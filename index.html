Mathematical proof

  • A mathematical statement is considered proven is we can deduce it from axioms and a formal system
  • Type Theory is a formal system

What's a type?

  • a type defines the set of values a term may have (eg. const x:boolean implies x can be true or false
  • a type defines the set of operations that can be applied to a term (eg. const not = (a: boolean):boolean => !a )

Kind

  • the type of type is often named kind
  • kind of number ? Type
  • kind of string ? Type
  • kind of Array ? Type => Type

Array<string> or Array<number> are types

Array is not a type, it's a type constructor

Dependent types

a type depends on a term


// Pseudo code
type Vect<n :number, A: Type> = A[] & {length: n}
